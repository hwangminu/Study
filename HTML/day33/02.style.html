단위
크기
  • % : 백분율
  • em : 배율
  • px : 픽셀
색상
  • HEX : #000000
  • rgb : rgb(255,128, 0)
  • rgba : rgba(255,255,192,0.5)
    red, green, blue, alpha(투명도)
  • colorName : lightgreen
    웹표준 고유 색상
가시 속성
display 화면에 보이는 형태 지정
visibility 보이거나 보이지 않거나
opacity 투명도 조절
박스 속성
width, height 영역의 너비와 높이를 지정
margin, padding 테두리 바깥 여백, 테두리 안쪽 여백
10px (all) : 상하좌우 모두 처리
10px auto (top-bottom, left-right) : 상하 , 좌우 각각처리
10px 5px 5px 10px (top right bottom left) : 상하좌우 서로 다르게 처리
box-sizing center-box : 영역의 크기는 가운데만 적용
border-box : 테두리를 포함한 영역을 적용
테두리 속성
border : 두께 유형 색상 테두리를 지정
border-radius 테두리의 둥글기를 지정
margin, padding 과 같은 방식으로 상하좌우 지정가능

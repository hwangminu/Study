display 값 특징
inline content 만큼의 크기를 차지한다. width, height를 적용할 수 있다.
크기의 요소가 중요하지 않을때 사용한다. (a태그, spen태그)
block 한줄을 혼자서 차지한다. width, height를 적용할 수 있다.
(div태그)
inline-block inline 처럼 한줄에 여러요소를 배치할 수 있고, 크기를 지정할 수 있다
예전에 사용하던 속성, 가~끔 사용한다. 장점만 모아놓은 속성
flex 내부 요소를 일정한 기준에 따라 정렬한다.
중심축, 교차축을 기준으로 내부요소들을 정리할때 사용
none 요소가 화면에서 사라진다. 자리차지도 없다
opacity 는 자리차지를 하지만, none 은 아예 사라진다.
